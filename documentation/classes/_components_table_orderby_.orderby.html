Class OrderBy

This pipe is used to make the table sortable by region, level or side.

 @Pipe({name: 'orderBy', pure: false})

Name: orderBy

Impure pipe: gets called every component change

Usage:

 *ngFor="let entry of data | orderBy: column"

sort the data by column, use '-' to reverse the sorting

  • transform(input: any, config?: string): any
    • Defined in components/table/orderby.ts:74

    Transform is an implementation of pipeTransform.

    Transform takes an array of inputs, sorts them based on the config ( + or - ), and returns a sorted array. If we only have one element in the input, no need to sort, just return the element.

    Parameters

    • input: any

      Input values.

    • Default value config: string = "+"

      Arg that states if we want to sort ascending or descending.
